## ✨ Key Features

### 🎬 Cinematic Presentation Scroll
A bespoke, native-overriding JavaScript scroll interceptor forces a strict **1-scroll = 1-section** behavior. Powered by a custom `requestAnimationFrame` loop utilizing an `easeOutQuint` physics curve for a blazing fast, yet buttery-smooth slide between "slides" (sections).

### 🏮 Atmosphere & Particle Engine
A multi-layered background particle system creates a living environment:
- **Chōchin (🏮)**: Subtly rising, softly glowing traditional lanterns.
- **Sakura (🌸)**: Gently falling cherry blossom petals with true 3D rotation physics.
- **Hotaru (✨)**: Jittery, pulsing fireflies that breathe life into the dark canvas.

### 📜 The "Hanko Stamp" Aesthetic
The UI embraces traditional Japanese print elements, including glowing deep-red borders, vertical Kanji typography (e.g., 献身 for Dedication), and subtle, washed-out watermark backdrops.

### 🎡 Interactive 3D Carousel
A fully interactive, infinitely scrolling horizontal 3D carousel built with Framer Motion natively handles the showcase of verified credentials and certificates inside glassmorphic cards.

### 🖼️ Premium Project Galleries
State-of-the-art cinematic modals for project showcases. Features full-screen blurring, dynamic real-estate utilization, and spring-animated gallery image sliders with dot indicators and live/repo dynamic linking.

## ⚙️ Local Development Setup

To run this portfolio locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/M7mdAyoub/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Experience the app**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
