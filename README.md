## ✨ Portfolio Engineering Highlights

This portfolio isn't just a static resume; it's a showcase of my front-end and systems capabilities, featuring:

### 🎬 Cinematic Presentation Scroll
A custom JavaScript scroll interceptor forces a strict **1-scroll = 1-section** behavior, leveraging a `requestAnimationFrame` loop and an `easeOutQuint` physics curve for a responsive, buttery-smooth transition between sections.

### 🏮 Atmosphere & Particle Engine
A multi-layered background particle system creates a living environment:
- **Chōchin (🏮)**: Subtly rising traditional lanterns.
- **Sakura (🌸)**: Gently falling cherry blossom petals with true 3D rotation physics.
- **Hotaru (✨)**: Jittery, pulsing fireflies that breathe life into the dark canvas.

### 📜 The "Hanko Stamp" Aesthetic
The UI embraces traditional Japanese print elements, including glowing deep-red borders, vertical Kanji typography (e.g., 献身 for Dedication), and subtle, washed-out watermark backdrops.

### 🖼️ Premium Project Galleries
State-of-the-art cinematic modals for my live projects. Features full-screen blurring, dynamic real-estate utilization, and spring-animated gallery image sliders with live data integration.
